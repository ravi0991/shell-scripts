# Shell Scripts

This repository contains **Bash scripts used for Linux and DevOps automation tasks**.

## Scripts Included

hello.sh – first shell script example  
variables.sh – using variables in Bash  
greet.sh – user input example  
check_number.sh – if-else conditions  
file_check.sh – file existence checks  
server_check.sh – check service status  
backup.sh – server backup automation  
log_rotate.sh – compress and clean old logs  
maintenance.sh – scheduled maintenance script

## Usage

Run a script using:

bash script-name.sh
